# Security

## Reporting

Report security concerns privately by emailing [dylan@wlim.work](mailto:dylan@wlim.work).

Do not open public issues, pull requests, or discussions containing secrets, tokens, customer data, private implementation details, unpublished routes, or reproduction steps that could expose private systems.

## Scope

This public repository contains product documentation only. It should not contain production source code, credentials, environment variables, private architecture, customer data, or proprietary workflows.

## No Public Disclosure Without Coordination

Please allow time for private review before sharing security-sensitive details publicly. No bug bounty program is implied by this file.
