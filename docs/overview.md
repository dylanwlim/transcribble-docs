# Transcribble Overview

Public guide for Transcribble, a transcription, decoding, and conversion tool for files.

## Live Link

## Status

Live public product surface.

## Product Scope

- Documents the public Transcribble surface and core user workflow.

## Primary Audience

People working with audio, transcript, decoding, and file-conversion workflows.

## Public Boundary

These docs describe only the public product surface. They do not publish private source code, credentials, private architecture, internal workflows, private routes, customer data, or implementation details.
