# Security And Privacy

Transcribble should be used with the same care you would use for any live website or app.

## Good Habits

- Treat audio, transcripts, names, meetings, and uploaded documents as sensitive.
- Use sample text when asking for help unless the exact content is necessary.
- If the Desktop Helper is part of your workflow, follow the setup prompts from inside the app.

## Reporting A Concern

For corrections or questions, email [dylan@wlim.work](mailto:dylan@wlim.work). If the concern involves account access, payment, health, portfolio, transcript, or client information, email first so it can be handled carefully.
